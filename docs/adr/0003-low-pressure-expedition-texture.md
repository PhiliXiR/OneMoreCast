# ADR 0003: Use expedition texture instead of survival or combat systems

**Status: Accepted**

One More Cast will create contrast around fishing through travel, passing time,
temporary route or weather inconvenience, and clearly explained fishing losses.
It will not introduce combat, hunger, or an always-draining survival meter as a
primary progression system. This preserves calm as something the player returns
to while keeping fishing, observation, and place learning as the core challenge.
