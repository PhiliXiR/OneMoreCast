# ADR 0002: Start with a compact foundation that can grow into a watershed

**Status: Accepted**

One More Cast will first ship and validate a compact, deeply authored fishing
RPG area rather than attempt a full watershed. The compact foundation must be
satisfying as a complete experience, while game-owned fishing, world, journal,
and progression systems use extensible patterns so later regions can join it
without rebuilding the core loop. This favors ecological density, memorable
places, and proven play over premature map scale.
